Apply for a building assessment certificate

Use this service to submit an application for a building assessment certificate.

The Building Safety Regulator will tell you when you need to apply for a building assessment certificate. You can only apply when BSR directs you to.

You will need to upload:

  • safety case report
  • mandatory occurrence reporting plan
  • resident engagement strategy

You will need to declare that you are keeping all of the relevant people, such as other accountable persons, up to date.

You will need to provide details so we can send you invoices. You will be charged for the work undertaken to assess your application.

You will also need to pay the application fee of £288. You can pay this by card or invoice.

Find out more about the Building Safety Regulator charging scheme.

Once you have submitted your application, it will take some time to review it. If we are not satisfied with your application, we will contact you to rectify any problems.

If you're granted a building assessment certificate, you will be able to download it. You must print it and display it somewhere in the building where residents will see it.
